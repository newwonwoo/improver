## 3. 제안 — 신규 기준 (출처·탐지난이도 명시)

> 핵심: 신규 기준 중 다수는 **기존 모호 룰보다 오히려 고정밀**(구조·사전 기반).

| 신규 코드 | 기준 | 출처 | 탐지방식 | 정밀도 전망 |
|---|---|---|---|---|
| **C-ARCHAIC** | 일본식·낡은 표현 | 법제처 정비기준 | 큐레이션 사전(당해→그, ~할 시→~할 때) | **높음** (사전 일치) |
| **C-DISCRIM** | 차별·권위적 표현 | 법제처 정비기준 | 큐레이션 사전 | **높음** |
| **C-CONFLICT** | 이해충돌(제척·기피·회피 부재) | 부패영향평가 | 위원회 조문 + 회피규정 부재 | 중상(구조) |
| **C-SUBSIDY** | 재정지원 기준 명확성 | 부패영향평가 | 보조금·지원 조문 + 지급기준 부재 + 위임 | 중상(구조) |
| **C-PRIVILEGE** | 특혜발생(자의적 자격·진입장벽) | 부패영향평가 | 자격요건 + 자의적 기준어 + 재량 | 중(다신호) |
| **C-BURDEN** | 준수부담(과도 구비서류) | 부패영향평가 | 제출·구비 항목 과다 + 위임 | 중 |
| **C-OVERSANCTION** | 과잉제재(비례 위반) | 부패영향평가/입안기준 | 경미위반 + 형사·취소 결합 | 중 |

## 4. 권고 실행 순서 (정밀도·근거 우선)
1. **C-ARCHAIC / C-DISCRIM** (일본식·차별 표현) — 법제처 사전이 **공식 확정 목록**이라
   큐레이션만 하면 고정밀. 사법확장 P-ARCHAIC scaffold 와 통합 가능. → 즉시 가치.
2. **C-CONFLICT / C-SUBSIDY** — 구조 신호(위원회/보조금 조문 + 특정 규정 부재)라
   기존 G-03 단문방식처럼 고정밀 가능.
3. **C-PRIVILEGE / C-BURDEN / C-OVERSANCTION** — 다신호 결합 필요, verdict 라벨 확보 후.

## 5. 정직한 한계
- 신규 기준도 **라벨·SME 검증 전엔 정밀도 미확정** — 측정 우선 원칙 유지.
- "더 많이"가 목표가 아니라 "근거 있고 고정밀하게 더" — 헛경보 룰 추가는 금지(이미 정리한 교훈).
- 의미판단 본질 기준(특혜·준수부담)은 키워드로 한계 → 구조 신호 + 향후 LLM 판정 병행.

## 6. C-ARCHAIC 실측 검증 (방법론 시연)
300개 법령 샘플 스캔:
- **naive 사전 매칭: 841건** → 그러나 "부의" 680건이 "일**부의** 정지/납**부의무**/정**부의**"
  등 조사 '의'의 부분일치 = **오탐**. (신규 기준도 naive하면 헛경보 — 기존 교훈 재확인)
- **형태소 경계·동사형 정밀 패턴: 111건** (87% 오탐 제거). 당해(57)·갈음(22)·제반(21)·
  해태(9)·부의 동사(2) 등 전부 법제처 공식 정비 대상어 → **진성 고정밀**.
- 전체 1,745개 법령 외삽 시 ~600건대 진성 C-ARCHAIC 결함 추정.

**결론**: 신규 기준은 (1) 권위 출처로 근거 있고, (2) 진성 결함이 코퍼스에 실재하며,
(3) **단, 반드시 정밀 큐레이션 필요**(naive 금지). 이 규율로 "근거 있고 고정밀하게 더"
확장하면 기준의 양·깊이 모두 향상.
