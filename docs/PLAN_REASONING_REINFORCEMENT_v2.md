## 1. 측정된 현재 위치 (재합의된 공정 잣대 V0')

같은 표본(n=2,588 row)·같은 fold(StratifiedKFold k=5 seed=42)·같은 micro-F1·
같은 bootstrap CI(1,000회)로 잰 값만 인정한다.

| 시스템 | total F1 | 95% CI | 출처 |
|---|---|---|---|
| 룰 단독 | 0.495 | [0.457, 0.530] | `outputs/v0prime_fair_baseline.json` |
| 선택적 앙상블(룰/SLM 카테고리 게이팅) | 0.551 | [0.515, 0.588] | `outputs/selective_ensemble_measure.json` |
| Torch MLP (61~74-dim 정량신호) | 0.468~0.516 | — | `outputs/torch_phase12/13_retrain.json` |

이미 측정으로 기각된 경로(반복 금지): 정량신호 추가(phase12·13 — 무효),
신호피처 통합(V4 — 무효), 카테고리 선택 앙상블 미세조정(+0.014 비유의),
MLP 과사양 가설(V3 — 기각).

## 2. 미시도 경로 (BRAIN_NN_ROADMAP 중 로컬·LLM 0회로 실행 가능한 것)

기존 시도는 전부 **61~74차원 정량신호**만 썼다. 조문 **텍스트 자체**는 한 번도
모델 입력이 된 적 없다(grep 확인: TF-IDF/boosting 구현 0건). 따라서:

- **E1 텍스트 단독**: char n-gram TF-IDF(2~4그램) + 카테고리별 LogisticRegression(balanced)
- **E2 부스팅**: 정량 dense 피처 + HistGradientBoosting (sklearn 내장, 외부 의존 0)
- **E3 결합**: TF-IDF ⊕ 표준화 dense → LogisticRegression
- **E4 선택 게이팅**: 카테고리별 {룰, E1, E2, E3, 룰∨모델} 중 train-fold 성능으로 선택
  (기존 selective ensemble과 동일한 누수 없는 선택 방식)

## 3. 측정 프로토콜 (감사인 게이트 — 위반 시 결과 무효)

1. **동일 잣대**: V0'와 같은 행·fold·micro-F1·bootstrap CI. 새 잣대 금지.
2. **누수 0**: TF-IDF·스케일러·모델·게이팅 선택 전부 train fold에서만 fit/결정.
   OOF(test fold) 예측만 집계.
3. **채택 기준**: total F1이 현행 최강(선택적 앙상블 0.551)을 초과해야 채택 논의.
   CI 겹침 여부를 명시하고, 비유의면 '비유의'로 기록(단정 금지).
4. **회귀 0**: 채택·통합 시 pytest 기존 통과분 유지.
5. **LLM·외부 API 0회.** 효과 없으면 '효과 없음'을 그대로 기록한다(V4·V6b 관행).
