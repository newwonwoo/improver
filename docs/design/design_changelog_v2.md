## 측정으로 확정된 잔여 블로커 (사람/라벨 의존)

- **자문위원 채택률(검수인 게이트 ③)**: 새 처방의 실무 채택 여부 = 사람 자문위원 블라인드 재검 필요. 증거 패키지 준비 완료.
- **fn(미발화) 검출 보강(코더)**: `outputs/fn_recall_baseline.json` — fn 22건 recall=0 재현, 진단 결과
  21/22가 '룰별 트리거 미스'(broad gate 아님). 신규 고정밀 트리거의 발화는 **라벨 부재**(미발화 조문엔
  verdict 없음)라 게이트①(precision 비악화) **검증 불가** → 신규 라벨링(사람/LLM) 선행 필수. 측정으로 확정.
