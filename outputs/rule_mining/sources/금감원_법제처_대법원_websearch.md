# 금감원·법제처·대법원 사례 종합 (WebSearch 수집)

## A. 금융감독원 — 검사·제재 사례

### A-1. DLF 불완전판매 (2019-2020)
- 5건 중 1건 불완전판매 의심
- 위반 유형:
  1. "원금손실 확률 0%" 강조 + 위험성 미설명
  2. 상품설명서 미교부
  3. 자필 누락 (직원 대필)
  4. 고령 투자자 절차 무시
  5. 무자격 직원 판매

### A-2. 라임·옵티머스·헤리티지 (2020-2022)
- **착오 취소 (100% 보상)**: 라임무역금융, 옵티머스, 헤리티지
- **부분 배상 (40~80%)**: 라임국내, 라임 CI, 디스커버리, 헬스케어
- 위반 유형:
  1. 거짓 투자제안서로 판매
  2. 펀드 돌려막기 사실 미고지
  3. 신분증 사본으로 임의 개설
  4. 자필 확인서 직원 대필
  5. 고령자 보호 절차 위반

### A-3. 홍콩 H지수 ELS (2024)
- 금감원 2024년 제재 절차 착수
- 위반 유형:
  1. 적합성 원칙 위반 + 설명의무 위반
  2. 고위험 상품 부적합 투자자에게 "가입하고 싶다" 발화 유도
  3. 지점 방문 어려운 투자자 대신 가입신청서 대리작성
  4. 녹취를 고객처럼 위장하여 허위 계약
  5. 과도한 KPI 설계로 영업 압박
- 제재 근거: 자본시장법·금소법 위반
- 제재 종류: 기관제재, 임직원 제재, 과징금·과태료

### A-4. GA(법인보험대리점) 위탁 규제 (2025.03)
- 보험사가 GA 위탁 시 검토 의무 항목:
  - 불완전판매율
  - 민원 발생률
  - 제재·금융사고 이력
  - GA 내부통제·지배구조
  - 소비자 보호체계
  - 변칙 영업행위 위험

### A-5. 보험판매 불완전판매 5대 유형
1. 적합성 위반: 투자성향 무시 권유
2. 설명의무 위반: 핵심 위험 누락
3. 부당권유: 만기 전 해지 권유, 사실 무관 수익률 보장
4. 자필 위반: 대필, 누락
5. 광고 규제 위반: 위험 미고지

## B. 법제처 — 법령해석 사례

### B-1. 위임 명확성
- 사실관계가 수시 변화하는 영역(예: 의무교육 실시) → 위임 명확성 요구 완화
- 일반 침익 영역 → 위임 명확성 엄격 적용

### B-2. 법령해석 제도
- 법제처가 행정부 내부 통일 해석 제공
- 민사·상사·형사, 행정소송, 국가배상관계법령은 제외
- 매년 법령해석사례집 발간 (data.go.kr/15083945)

### B-3. 행정입법 한계
- 위임범위 일탈 시 위법 (대법원 다수 판례)
- 시행령이 법률 위임 없는 새 의무·제한 부과 → 무효

## C. 대법원 행정판례 패턴 (2023-2024)

### C-1. 재량권 일탈·남용 심사기준 (행정소송법 §27)
대법원이 재량처분을 취소하는 4가지 기준:
1. **사실오인**: 처분 전제 사실의 오류
2. **비례원칙 위반**: 목적-수단 비례성 결여
3. **평등원칙 위반**: 합리적 이유 없는 차별
4. **목적 위반·부정한 동기**: 본래 목적과 다른 의도

### C-2. 주요 판례 (2024.07.11)
- 개발제한구역 내 조명탑 원상복구 처분 → 재량남용 인정
- 행위허가구역 경계 벗어난 지점에 조명탑 설치 사안
- **시사점**: 형식적 위반에 대한 최대 제재 → 비례원칙 위반

### C-3. 처분 이유제시 의무 (행정절차법 §23)
- "이유를 제시하지 않은 처분"은 절차상 하자
- 거부처분·취소처분에 적용
- 처분 사유의 구체적 명시 요구

### C-4. 입법 결함으로 귀결되는 판례 패턴 (JUD 시리즈)
| 룰 ID | 판례 패턴 | 입법 결함 |

| JUD-01 | 1:1 자동 제재 (가중·감경 없음) | 비례성 결여 |
| JUD-02 | 거부 사유 열거 없음 | 이유제시 우회 |
