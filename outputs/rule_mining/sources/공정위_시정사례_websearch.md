# 공정거래위원회 약관·시정명령 사례 종합 자료집
> WebSearch (2026-05-22) 으로 정부 사이트 우회 수집
> 룰 마이닝 프롬프트에 첨부 시 활용 권장

## 1. 은행·저축은행 (2025.10.29)
- 총 1,735개 약관 심사 → 60개 약관 (17개 유형) 시정 요청
- 은행 56개 (14개 유형), 저축은행 4개 (3개 유형)
- **유형**:
  1. 자의적 서비스 변경·중단 ("기타 필요한 경우" 등)
  2. 부적절한 통지 방식 (개별 통지 생략, 부적절한 방식)
  3. 일방적 급부 결정권
  4. 전산장애 등 사업자 귀책 전면 면제
  5. 전자금융 예금 해지 과도 제한

## 2. 가상자산 (2021.07.29)
- 8개 가상자산사업자 → 15개 유형 시정 권고
- 유형: 약관 개정권, 이용계약 중지·해지권, 서비스 이용제한, 부당 면책

## 3. OTT 플랫폼 (2021)
- 6개 사업자 (넷플릭스·웨이브·티빙·시즌·왓챠·구글) → 7개 유형
- 유형:
  1. 중도 해지 환불 거부
  2. 부당한 위약금
  3. 청약철회권 제한
  4. 자의적 요금 변경 (사전 동의 없음)
  5. 현금 환불 거부 (예치금으로 환급)
  6. 선물 사이버머니 환불불가
  7. 회원 계정 종료 사유 불명확
- **넷플릭스 개선**: 결제 주기 내 환불 불가 → 사업자 귀책 또는 미이용 시 7일 이내 환불 가능

## 4. 오디오북 (2024)
- 5개 사업자 → 3개 유형
- 유형: 이용내역 없는데 환불 제한 / 예치금 환급 강제 / 무료 → 유료 자동 전환

## 5. 게임 모바일 (10개 사업자, 14개 유형)
- 아이템 선물 청약철회 가능
- 청약철회 제한 (전자상거래법 외 사유로 제한 불가)
- 위법행위 배상권 보장
- 이용자 제재 시 사전 통지 의무
- 게임 내 교신내용 무제한 열람 금지
- 미성년자 법정대리인 개별 동의 (포괄 동의 불가)

## 6. 결혼식·예식업
- **24개 예식장**: 계약해제 시점 무관 계약금 환불 거부 조항 → 예식일 90일 전 통보 시 전액 환급 가능
- **18개 결혼준비대행업체** (6개 유형 시정)

## 7. 공연·티켓 (2024-2025, 19개 사업자)
- 4개 분야 9개 유형 시정
- **대상**: 예술의전당, 롯데콘서트홀, 부산문화회관, 국립국악원, 인터파크, 클럽발코니 등
- 유형:
  1. 부당한 환불 제한 (일부 혜택 이용 → 연회비 전액 환불 거부) → 14~30일 내 전액 환불, 사용 시 합리적 위약금만 공제
  2. 사업자의 부당한 면책 (고의·과실 손해 책임 회피)
  3. 이용자 권리행사 제한 (게시물 일방 삭제, 전화로만 탈퇴)
  4. 기타 불공정

## 8. 헬스장 표준약관 (2024)
- 휴·폐업 14일 전 통지 의무화

## 9. 신용카드·통신
- **할부항변권**: 7일 내 철회 가능 (할부거래법 위반 시정)
- **통신사 약정**: 번호이동 시 사전 고지 없는 위약금 청구 → 취소·환불

## 10. 대리점거래 (2024)
- 약식2024-031, 약식2024-047 미지급금 시정
- 대리점계약서 개정, 판매장려금 지급 시정
