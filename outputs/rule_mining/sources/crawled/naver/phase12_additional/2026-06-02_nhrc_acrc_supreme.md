## D. 신호·규칙 검증 매트릭스 (이번 추가 100건)

| 우리 신호/규칙 | 이번 추가 일치 사례 | 누적 매칭 (Phase 12 전체) |
|---|---|---|
| R-DISPROPORTIONATE (비례 위반) | 5건+ (유승준 3건 + 음주운전 + 학교용지) | 신뢰도 ↑ |
| R-NO-HEARING (절차 부재) | 3건 (서울대 교수, 살인범, 재판소원) | 신뢰도 ↑ |
| R-NO-REASON (이유 부재) | 2건 (서울대 교수, 박찬욱) | 신뢰도 유지 |
| R-DISP-ARBITRARY (자의적) | 4건 (육아휴직, 마이스터고, 헬스장, 콜마비엔에이치) | 신뢰도 ↑↑ |
| has_age_discrimination | 3건 (헬스장 17세, 마이스터고, 임신·출산) | 신호 검증 ✓ |
| R-NO-DISP-STANDARD (신규) | 2건 (요양병원 광범위 해석, 권익위 권고 거부 가능) | n 작음 |
| R-SUBDELEG-ADMIN-RULE (신규) | 1건 (재난적의료비 내부지침 — 이미 수집) | n 작음 |

**핵심 통찰**:
1. **재량권 일탈·남용**은 한국 행정소송의 가장 빈번한 쟁점 — R-DISPROPORTIONATE 의 신뢰도 prior 를 0.50 → 0.60 으로 상향 검토 필요
2. **인권위 평등권 침해 차별행위** 결정례는 대부분 자의적 기준 적용 → R-DISP-ARBITRARY 의 적용 폭 확장 (사회 정책 영역까지)
3. **대법원 → 행정청 거부 반복 사례** (유승준)는 R-NO-REASON 및 비례 원칙 결합 검증 케이스
