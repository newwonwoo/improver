## 자동 다운로드 불가 자료 (사용자 수동 다운로드 필요)

샌드박스 네트워크 정책상 `law.go.kr / ftc.go.kr / bai.go.kr / data.go.kr / namu.wiki` 등은 모두 차단되어 자동 다운로드가 불가능합니다.

아래 자료는 사용자가 직접 다운로드 후 `sources/external/` 에 넣어 주세요. (이미 룰 마이닝 프롬프트에는 핵심 요약이 포함되어 있어 옵션입니다)

### A. 공정거래위원회 — 약관심사지침
- URL: <https://www.law.go.kr/행정규칙/약관심사지침>
- 형식: 행정규칙 (HWP/PDF)
- 내용: 불공정약관 9개 유형 + 시정명령 사례

### B. 감사원 — 감사결과 처분요구 공개자료
- URL: <https://www.bai.go.kr/bai/result/branch/list/>
- 형식: 처분요구서 PDF (연 200~300건)
- 추천: 규제·입법 관련 키워드 필터링 (위임·처분·과징금)
- **대안**: 이미 행정기본법·행정절차법 원문이 있어 핵심 패턴은 추론 가능

### C. 금융감독원 — 금융기관 검사 및 제재에 관한 규정 시행세칙
- URL: <https://www.law.go.kr/LSW/admRulInfoP.do?admRulSeq=2200000106255>
- 형식: 행정규칙
- 내용: 금융기관 검사 절차·제재 사유 상세

### D. 법제처 — 법령입안심사기준 2024
- URL: <https://www.data.go.kr/data/15031101/fileData.do>
- 형식: PDF 책자 (~500p)
- 내용: 입안심사 항목 전체

### E. 국민권익위원회 — 고충민원 결정례집 2024
- URL: <https://www.data.go.kr/data/15053340/fileData.do>
- 형식: PDF 책자
- 내용: 분야별 결정례 + 법령개선 권고 사례

### F. 대법원 행정판례
- URL: <https://glaw.scourt.go.kr/>
- 검색어: "재량권 일탈·남용", "처분 이유제시", "비례원칙"
- 추천: 최근 10년 주요 판례 30~50건

## Claude.ai 웹 활용 안내

1. `_PROMPT_GUIDE.md` 의 우선순위대로 `01_*.md` ~ `06_*.md` 를 Claude.ai 에 붙여넣기
2. 더 정확한 응답을 원하면, 각 프롬프트 끝에 위 sources/*.md 파일 내용을 추가 첨부
3. 응답 JSON 을 `outputs/rule_mining/responses/<bundle_id>.json` 으로 저장
4. (예정) import 스크립트가 자동으로 새 룰을 `engine/rules/` 에 생성
