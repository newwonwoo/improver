# 금융감독원 금융소비자보호법 위반 유형 전수 룰 마이닝

당신은 한국 법령 분석 AI 엔진의 규칙 설계자입니다.
금융소비자보호법(이하 금소법) 및 금감원 검사·제재 패턴을 분석하여
아직 구현되지 않은 모든 규제 결함 룰을 설계해 주세요.

---

## 현재 구현된 룰 (제외)

- **F-09** (FSS-01·FSS-05): 설명의무·계약서 교부의무 부재 → 이미 구현

---

## 금융소비자보호법 주요 규제 의무 체계

### FSS-02: 적합성 원칙 위반 (금소법 §17)
금융상품 판매 시 소비자의 재산상황·투자경험·목적에 맞지 않는 상품을 권유하는 조항.
- TP: 금융상품 판매·권유 조항 + 소비자 적합성 확인 절차 없음
- TP: "투자자 성향 관계없이 판매 가능" 형태 (적합성 우회)
- TP: 고위험 상품 + 일반 소비자 대상 + 보호 조치 없음
- FP: 기관투자자·전문투자자 대상 상품 (적합성 원칙 적용 제외)

### FSS-03: 적정성 원칙 위반 (금소법 §18)
소비자가 자발적으로 구매하는 경우에도 부적정 상품임을 알리지 않는 조항.
- TP: 비보호 금융상품(파생상품 등) + 적정성 고지 의무 없음
- TP: "구매 요청 시 즉시 판매 가능" + 위험 고지 없음
- FP: 원금 보장 상품, 예금·적금류

### FSS-04: 불공정 영업행위 금지 위반 (금소법 §20)
소비자에게 부당하게 불리한 영업 방식을 강제하는 조항.
- TP: "대출 실행 조건으로 보험 가입 강제" (끼워팔기)
- TP: "서비스 이용 조건으로 개인정보 동의 강제"
- TP: 불이익 제공으로 금융상품 계약 강요 조항
- FP: 패키지 상품 (약관에 명시, 소비자 선택 가능)

### FSS-05: 부당 권유 행위 금지 위반 (금소법 §21)
소비자 이익에 반하는 금융상품을 권유하는 조항.
- TP: 만기 미도래 계약 해지 권유 + 새 계약 체결 강요
- TP: 소비자 위험 성향 무시한 고위험 상품 강권
- TP: 사실에 근거하지 않은 수익률 보장 약속 조항
- FP: 소비자 요청에 따른 상품 안내

### FSS-06: 광고 규제 위반 (금소법 §22)
금융상품 광고에서 소비자를 오인시킬 수 있는 내용을 규정한 조항.
- TP: 원금 손실 가능성 미고지 + 수익률만 강조하는 조항
- TP: 과거 수익률이 미래 수익 보장이 되는 것처럼 표현
- TP: 보험료·수수료 등 비용 미고지 조항
- FP: 단순 상품 소개 (구체적 수익률 언급 없음)

### FSS-07: 분쟁조정 전치 의무 회피 (금소법 §36)
금융 분쟁 시 분쟁조정 신청을 방해하거나 우회하는 조항.
- TP: "소비자 동의 없이 소송 제기 가능" 약관
- TP: 분쟁조정 신청 전 소송 제기 강제
- TP: 소비자의 분쟁조정 신청권 포기 동의 조항
- FP: 기업 간 계약 (B2B), 전문투자자 간 거래

### FSS-08: 청약 철회권 제한 (금소법 §46)
소비자의 금융상품 청약철회권을 배제하거나 제한하는 조항.
- TP: "계약 체결 후 청약철회 불가" 조항
- TP: 법정 청약철회 기간(15일)보다 짧은 기간 설정
- TP: 청약철회 시 과도한 위약금 부과
- FP: 청약철회 자체가 불가능한 상품 유형 (이미 이행된 서비스 등)

## 금감원 검사·제재 주요 결함 유형 (최근 10년 빈출)

### 불완전판매 3대 패턴
1. **설명서 미교부**: 계약 체결 시 상품설명서 미제공
2. **핵심 위험 누락 설명**: 원금손실 가능성, 중도해지 불이익 미고지
3. **서명 날인 없는 계약**: 소비자 확인 절차 없이 계약 체결

### 내부통제 부재 패턴
- 판매 적합성 심사 절차 없음
- 불완전판매 사후 모니터링 부재
- 해피콜(완전판매 확인) 형식화

## 요청

위 FSS-02~FSS-08 및 불완전판매 패턴 **전부를 룰로 설계**하세요.
F-09(설명의무·교부의무)와 중복되지 않게 설계하세요.

**JSON 형식으로 출력**:

```json
{
  "rules": [
    {
      "rule_id": "FSS-02",
      "rule_name": "적합성 원칙 미비 — 소비자 투자성향 확인 절차 없음",
      "category": "공정성",
      "source": "금융소비자보호법 제17조",
      "tp_signals": [
        "금융상품 판매·권유 조항 + 투자자 성향/재산 확인 절차 없음",
        "일반 소비자 대상 고위험 상품(파생상품·ELS 등) + 보호 조치 없음"
      ],
      "fp_filters": [
        "기관투자자·전문투자자 전용 상품 (적합성 원칙 제외)",
        "단순 예금·적금 상품"
      ],
      "regex_hints": [
        "투자\\s*권유.{0,100}(?:확인|평가|점검)\\s*없",
        "(?:파생상품|ELS|DLS).{0,100}판매\\s*(?:가능|허용)"
      ],
      "severity": "경고",
      "description": "금융상품 판매·권유 시 소비자의 투자 목적·재산 상황·투자 경험을 확인하는 절차 없이 판매 가능하도록 규정 (금소법 §17 위반)",
      "detection_scope": "금융업 관련 법령만 적용 (은행법, 자본시장법, 보험업법 등)"
    }
  ]
}
```

**주의**: FSS-02~FSS-08 모두 설계 (총 7개 유형). 금융업 법령에서만 적용되도록 detection_scope 명시 필수.
