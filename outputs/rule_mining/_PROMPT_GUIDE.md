# 룰 마이닝 프롬프트 가이드

Claude.ai 웹에 각 파일을 그대로 붙여넣어 주세요.
응답을 `outputs/rule_mining/responses/` 폴더에 저장하면 됩니다.

## 파일 목록 (우선순위 순)

| 파일 | 출처 | 예상 신규 룰 |
|------|------|------------|
| `01_ftc_약관규제법_전수.md` | 공정위 약관규제법 §6~§14 (9개 조항) | FTC-01~FTC-09 |
| `02_bai_감사원_규제입법.md` | 감사원 감사원칙 + 처분요구 유형 | BAI-01~BAI-10 |
| `03_fss_금소법_판매규제.md` | 금감원 금융소비자보호법 위반 유형 | FSS-01~FSS-08 |
| `04_moleg_법제처_입안심사.md` | 법제처 법령입안심사기준 (2024) | MOL-01~MOL-12 |
| `05_acrc_권익위_고충민원.md` | 권익위 고충민원 → 법령개선 권고 패턴 | ACRC-01~ACRC-08 |
| `06_court_대법원_행정판례.md` | 대법원 재량권·처분이유 관련 판례 패턴 | JUD-01~JUD-06 |

## 응답 포맷

Claude 웹에게 다음 JSON 형식으로 출력하도록 요청합니다:
```json
{
  "rules": [
    {
      "rule_id": "FTC-03",
      "rule_name": "손해배상액 예정 — 소비자에게 과중한 배상의무",
      "category": "공정성",
      "source": "약관규제법 제8조",
      "tp_signals": ["배상액", "위약금", "손해배상", "..."],
      "fp_filters": ["사업자→소비자 배상", "..."],
      "regex_hints": ["배상액.*정한다", "위약금.*%"],
      "severity": "경고",
      "description": "..."
    }
  ]
}
```
