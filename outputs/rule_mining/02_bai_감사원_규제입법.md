## 감사원 주요 감사 결과 처분요구 패턴 (규제입법 분야)

### BAI-02: 이중제재 — 과태료+형사벌 병과
**근거**: 헌법 제13조, 이중처벌금지 원칙, 대법원 2003두1830
같은 위반행위에 과태료(행정제재)와 형사처벌을 동시에 부과하는 조항.
특히 경미한 의무 위반에 과태료+징역·벌금 병과 → 비례원칙 위반.
- TP 예시: "제00조를 위반한 자는 3년 이하의 징역...과태료 000만원"
- TP 예시: 동일한 행위가 벌칙 조항과 과태료 조항 양쪽에 동시 규정

### BAI-03: 청문·의견제출 절차 누락
**근거**: 행정절차법 제22조(의견제출), 제24조(청문), 제26조(처분의 이유제시)
침익적 행정처분(허가취소·영업정지·과징금 등) 조항에서 청문·의견제출 절차가 없는 경우.
- TP 예시: "허가를 취소할 수 있다" + 청문 절차 규정 없음
- TP 예시: "영업을 정지할 수 있다" + "다만 청문을 실시하여야 한다" 부재
- FP 필터: 긴급처분 명시 ("긴급한 필요가 있는 경우"), 경미한 처분

### BAI-05: 행정지도의 강제화
**근거**: 행정절차법 제48조(행정지도 원칙), 행정지도는 임의협력 원칙
행정지도(권고·조언·지도)를 사실상 강제 의무로 만드는 조항.
- TP 예시: "지도·감독할 수 있다" + "이에 따르지 않으면 과태료"
- TP 예시: "권고를 따르지 아니한 경우 영업 정지" 형태
- FP 필터: 실질적 행정처분 조항 (허가·인가·등록 등)

### BAI-06: 고시·훈령으로의 실질적 위임
**근거**: 법률유보원칙, 헌법 제75조
법률 위임 없이 고시·훈령·예규·지침으로 국민 권리·의무를 규율하는 조항.
- TP 예시: "세부 기준은 고시로 정한다" + 침익적 처분 기준 위임
- TP 예시: "운영지침에서 정하는 바에 따른다" (법적 구속력 없는 행정규칙으로 위임)
- FP 필터: 서식·수수료 등 순수 절차 사항, 기술 기준(성능 요건 등)

### BAI-07: 이해충돌 방지 규정 부재
**근거**: 이해충돌방지법 제5~14조, 공직자윤리법
인허가·심의위원회 조항에서 위원의 이해충돌 방지·제척·기피 절차가 없는 경우.
- TP 예시: "심의위원회를 둔다" + 제척·기피·회피 규정 없음
- TP 예시: "위원회 위원은 관련 업계 전문가를 포함한다" + 이해충돌 방지 없음
- FP 필터: 자문기구 (의결권 없는 순수 자문), 이미 타 법령에서 규율

### BAI-08: 처분기준 미공표
**근거**: 행정절차법 제20조 (처분기준 사전공표 의무)
재량적 행정처분을 규정하면서 처분 기준을 사전에 정해 공표하도록 하는 규정이 없는 경우.
- TP 예시: "취소할 수 있다" + "처분기준은 대통령령으로 정한다" 없음
- TP 예시: 과징금·행정처분 조항 + 세부 기준 미명시
- FP 필터: 기속행위 (재량 없는 조문), 이미 하위법령에서 기준 규정

### BAI-09: 처분 이유제시 의무 회피
**근거**: 행정절차법 제23조 (처분 이유제시), 대법원 2019두31839
침익적 처분 조항에서 이유제시 의무를 명시하지 않거나, 간이절차로 우회하는 조항.
- TP 예시: "즉시 취소할 수 있다" (이유제시 없이 즉시 처분)
- TP 예시: "사전 통보 없이 처분 가능하다" 형태
- FP 필터: 긴급 공익상 필요가 명시된 경우

### BAI-10: 환경·안전 기준 위임 시 구체성 결여
**근거**: 환경정책기본법, 산업안전보건법, 식품안전기본법
환경·안전 기준을 하위법령으로 위임하면서 기준 설정의 원칙·방향·한도를 전혀 제시하지 않는 조항.
- TP 예시: "안전 기준은 환경부령으로 정한다" (상한·하한·원칙 없음)
- TP 예시: "배출기준은 고시로 정한다" + 기준치 범위 미제시

## 행정절차법 기반 추가 결함 유형 (감사원 지적 빈도 높음)

### APL-01: 처분 전 사전통지 누락
행정절차법 제21조 — 침익처분 전 당사자에게 사전통지 의무.
"사전 고지 없이 처분 가능" 또는 사전통지 절차가 아예 없는 조항.

### APL-02: 처분기간 미명시
행정절차법 제19조 — 처리기간 설정·공표 의무.
인허가·처분 조항에 처리 기간(몇 일 이내)이 전혀 명시되지 않은 경우.

### APL-03: 의견청취 결과 반영 의무 부재
행정절차법 제27조의2 — 의견 제출 후 반영 여부 통보 의무.
의견제출 절차는 있지만 반영 여부 통보 의무 없는 조항.

## 요청

위 BAI-02~BAI-10, APL-01~APL-03 유형 중 **L-04, L-06으로 커버되지 않는 모든 유형**을 룰로 설계하세요.

**각 룰에 대해 다음 JSON 형식으로 출력**하세요:

```json
{
  "rules": [
    {
      "rule_id": "BAI-02",
      "rule_name": "이중제재 — 동일행위 과태료·형사벌 병과",
      "category": "적법성",
      "source": "헌법 제13조, 이중처벌금지 원칙",
      "tp_signals": [
        "같은 조에서 형사처벌 + 과태료 동시 규정",
        "위반 시 징역·벌금 + 별도 과태료 중복 부과"
      ],
      "fp_filters": [
        "행정범과 형사범 명확히 구분된 경우",
        "경과실·의도 구분하여 다른 제재 부과"
      ],
      "regex_hints": [
        "이하의\\s*(?:징역|벌금).{0,200}과태료",
        "과태료.{0,200}이하의\\s*(?:징역|벌금)"
      ],
      "severity": "경고",
      "description": "동일한 위반행위에 과태료(행정제재)와 형사처벌을 동시에 부과하는 이중제재 조항",
      "detection_scope": "단일 법령 내 벌칙 조항과 과태료 조항 크로스 체크 필요"
    }
  ]
}
```

**주의사항**:
- 한국 법령 텍스트에서 실제로 등장하는 표현으로 regex_hints 작성
- FP가 많은 유형은 fp_filters를 상세히 작성
- detection_scope가 단일 조문인지 법령 전체 스캔인지 명시
- BAI-02~BAI-10, APL-01~APL-03 모두 커버 (총 13개 유형)
